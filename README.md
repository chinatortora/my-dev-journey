# 🚀 Multidisciplinary Dev Showcase: Python, SQL & APIs

Welcome! This repository serves as a portfolio of my technical skills, ranging from fundamental programming logic to professional API architecture and database design.

---

## 📂 Repository Structure

### 1. 🌸 Pink Pokemon Power (Console Game)
A logic-based interactive application that demonstrates:
* **API Consumption:** Using `requests` to fetch JSON data from PokéAPI.
* **Data Structures:** Advanced use of lists and dictionaries to filter and process information.
* **File Management:** Automated logging of game results and data exports.
* **Logic:** Efficient use of functions, loops, and string slicing.

### 2. 🏋️‍♂️ Fitness Center Database (MySQL)
A relational database project focused on business intelligence:
* **Schema Design:** Tables organized for practitioners, activities, and memberships.
* **Complex Queries:** Logic to identify birthday rewards, multi-enrollment discounts, and age demographics.

### 3. 📅 Fitness Class API (Flask Architecture)
A professional-grade API implementation following the principle of **Separation of Concerns**:
* **Layered Design:** Split into `main` (Client), `app` (Server), and `db_utils` (Database interaction).
* **Security:** Use of a `config` file to protect sensitive database credentials.
* **REST Standards:** Clear endpoints for viewing and registering for classes.
